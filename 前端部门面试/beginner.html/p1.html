数学经典问题——鸡兔同笼问题
在c语言上的计算实现



问题简介：
《孙子算经》 中“鸡兔同笼”问题原文如下： 今有雉兔同笼，上有三十五头，下有九十四足，问雉兔各几何？
翻译成现代白话文为： 现在有若干只鸡和兔在同一个笼子里，从上面数，有35个头；从下面数，有94只脚。

查看题解 

                
                      #include 
                    #include 
                        int main()
                        {
                        int m,n;
                        while(scanf("%d %d",&n,&m)!=EOF){
                        if(n==0 && m==0){
                        return 0;
                        }
                        int x,y;
                        y=(m-n*2)/2;
                        x=n-y;
                        if(x+y==n &&x*2+y*4==m&&x>0&&y>0){
                        printf("%d %d\n",x,y);
                        }else {
                        printf("Error\n");
                        }
                        }
                        return 0;
                        }

题解

首先输入n个头，m只脚.然后定义俩个变量x，y
m-2*n为“多”出的脚数,以此计算出兔子的数目
二者相减 即可得出鸡的数目。
判断给出的m，n是否成立在输出x，y。

按照《孙子算经》所述题目，n=35，m=94.
